# Shivu's Cybersecurity Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 How to Go Live (GitHub Pages)

### Step 1 — Create GitHub repo
1. Go to github.com → New Repository
2. Name it exactly: `yourusername.github.io`
   (Replace "yourusername" with your actual GitHub username)
3. Make it **Public**
4. Click **Create Repository**

### Step 2 — Upload your files
Option A (Drag & Drop):
1. Open the repo on GitHub
2. Click "uploading an existing file"
3. Drag all 3 files: `index.html`, `style.css`, `script.js`
4. Click "Commit changes"

Option B (Git commands):
```bash
git init
git add .
git commit -m "Initial portfolio"
git remote add origin https://github.com/yourusername/yourusername.github.io
git push -u origin main
```

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Source: Deploy from branch → main → / (root)
3. Click **Save**

### Step 4 — Your site is live!
URL: `https://yourusername.github.io`
(Takes 1-2 minutes to go live)
